CCTV 연계 스마트 주차 관리 시스템

기술 스택: Python, OpenCV, React, Node.js, MongoDB, IoT (Raspberry Pi, CCTV), Tesseract (OCR)

이 시스템은 **CCTV 카메라**와 연계되어 주차 공간의 상태를 실시간으로 모니터링하고, **자동 번호판 인식** 기술을 활용하여 차량 출입을 관리하는 시스템입니다. 주차 공간 예약과 실시간 위치 파악, 출입 기록 관리 등을 통해 **스마트 주차 시스템**을 구현합니다. 이 시스템은 주차 공간을 보다 효율적으로 사용할 수 있게 도와주며, 안전한 주차 환경을 제공합니다.

기술 스택

저는 다양한 최신 기술을 사용하여 프로젝트를 진행하고 있습니다.

  • 프론트엔드: React를 사용하여 직관적인 사용자 인터페이스를 개발하고 있습니다.
  • 백엔드: Node.js와 Express로 API 서버를 구축하고 데이터 처리 로직을 담당하고 있습니다.
  • 데이터베이스: MongoDB를 사용하여 데이터를 효율적으로 관리하고 저장합니다.
  • 이미지 처리: OpenCV와 YOLO를 사용하여 **자동 번호판 인식** 및 **차량 감지** 기술을 구현했습니다.
  • IoT: Raspberry Pi와 Arduino를 사용하여 **스마트 장치**와의 연동을 실현했습니다.
